# body-brain-coevolution
Co-evolving virtual creature morphology and neural controller in Unity using the Python DEAP library.

Steps to run:

1. Run backend
cd python_backend
./run_backend.sh

2. Python endpoints
Navigate to http://localhost:8000/docs
get_genome -> Try it out -> Execute

3. Unity project
Open project stored in /unity_sim/body-brain-coevolution-unity/
Run project
Watch for correspondence between Unity project console and terminal running Dockerfile
